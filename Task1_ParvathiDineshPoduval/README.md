# Task 1: Rule-Based Chatbot

## Project Description

This project is a simple **Rule-Based Chatbot** developed using Python. The chatbot responds to predefined user inputs using conditional statements (`if`, `elif`, and `else`). It continuously interacts with the user until an exit command is entered.

Unlike machine learning chatbots, this chatbot follows predefined rules to generate responses, making it a great introduction to basic Artificial Intelligence concepts.

---

## Objective

Develop a chatbot that can:

- Respond to greetings
- Answer predefined questions
- Handle unknown inputs gracefully
- Exit the conversation when requested by the user

---

## Features

-  Responds to greetings such as **Hello**, **Hi**, and **Hey**
-  Answers common questions like **How are you?**
-  Introduces itself when asked
-  Displays the current date and time
-  Performs simple calculations (Addition and Multiplication)
-  Answers basic AI-related questions
-  Handles invalid or unknown inputs
-  Runs continuously until the user enters **bye**, **exit**, or **quit**
